## React로 만드는 mytube입니다.

기존에 만들었던 mytube는 백엔드(nodejs,mongodb)와 프론트(pug)로 만들었다면

이번 mytube_react는 유튜브 API를 이용해 리액트를 익히기 위함입니다.

React Hook과 PostCSS 를 이용하였습니다.
